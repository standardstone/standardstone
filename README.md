# face-recognition
## 프로젝트 개요
opencv, dilb,face-recognition을 활용하여 영상,사진,웹켑에서 얼굴인식이 가능하게 만들었다.
얼굴인식을 영화 수리남에 활용하여 극중 쥐새끼인 배우 유인석을 다른 배우들과 구분하여 Mouse로 찾게 만들었다.
